# Run the project

1) Create a `.env` file in the project root with these variables:

VITE_CHATFLOWID=26d64e29-3a3f-4da2-97a0-637ba55b62b5
VITE_API_HOST=https://adp-537046-stage.1ai.my.id

2) Install dependencies:

```
npm install --legacy-peer-deps
```

3) Start the development server:

```
npm run dev
```

4) Open the URL shown by Vite (usually http://localhost:5173).

Demo

Try the included demo prompts in the repo:

- prompt-html.md — HTML demo output
- prompt-mermaid.md — Mermaid diagram demo

Open those files and paste their contents into the app prompt/input to run the demos.
